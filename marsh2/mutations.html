Customized Mutation Group

Upload your own text file with amino-acid level mutations to calculate enrichment at protein-protein interaction interfaces across the human interactome.

Uniprot Accession Identifiers are required.

Input one mutation per line, separated by a single space or tab. For example: 
                P40692 640 P T 
                P40692 653 L R
Please use single-letter notation for Amino-Acids. You can download a sample file here.

Once you've submitted your set of mutations you will receive a personal mutation code , which will be your unique identifier to access this private set of mutations on our site.
